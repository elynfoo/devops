Simple Web Scanner Web App

Tech Stack:

  • Operating System: Ubuntu/Debian Linux (VirtualBox VM)
  • Web Server: Apache2 → serves PHP dashboard and scan results
  • Scripting Language: PHP → formats and displays Nmap output
  • Network Scanner: Nmap → scheduled scans of network range
  • Automation: Cron → triggers Nmap scan script every 5 minutes
  • File & Permissions: chmod, chown, groups → secure file ownership
    Avoids insecure chmod 777 by using group-based permissions (e.g., 775)

Workflow Overviewk:

  1. Cron job runs every 5 minutes → executes network_scan.sh
  2. Nmap scans → outputs results to /var/www/html/output.txt
  3. Apache + PHP serve network.php → formats output file
  4. Browser dashboard shows updated scan results
